# Oscars App

- Relational database project themed around the Oscars: designed the entity relationship diagram and normalized a large dataset containing historical Oscar nominations into SQL tables
- Python Flask/Jinja web app with SQLite, allowing users to view the entity relationship diagram, explore tables, and run SQL queries of increasing complexity
